Selectores básicos

Los selectores básicos en CSS te permiten seleccionar elementos en tu documento HTML para aplicarles estilos.

  • Selector de tipo: selecciona elementos por su nombre de etiqueta (ejemplo: p, h1, div).
  • Selector de clase: selecciona elementos por su atributo de clase (ejemplo: .mi-clase).
  • Selector de ID: selecciona un elemento por su atributo de ID (ejemplo: #mi-id).

    Selectores universales y combinaciones

    Los selectores universales y combinaciones en CSS te permiten seleccionar elementos de manera más específica y flexible.

  • Selector universal: selecciona todos los elementos en el documento (ejemplo: *).
  • Combinaciones de selectores: selecciona todos los elementos en el documento (ejemplo: *).
  • Combinador descendente: selecciona elementos que son descendientes de otro elemento (ejemplo: div p).
  • Combinador hijo: selecciona elementos que son hijos directos de otro elemento (ejemplo: div > p).
  • Combinador adyacente: selecciona un elemento que es adyacente a otro elemento (ejemplo: h1 + p).
  • Combinador general de hermanos: selecciona elementos que son hermanos de otro elemento (ejemplo: h1 ~ p).

    Display y tipos de cajas

    El modelo de caja en CSS describe cómo se representan los elementos en la página web. Cada elemento se representa como una caja rectangular que consta de varias partes: el contenido, el relleno, el borde y el margen.

  • Contenido: es la parte central de la caja donde se encuentra el contenido del elemento.
  • Relleno (padding): es el espacio entre el contenido y el borde de la caja.
  • Borde (border): es la línea que rodea la caja y puede tener diferentes estilos, colores y grosores.
  • Margen (margin): es el espacio entre la caja y otros elementos en la página.

    Pseudo-clases

    Las pseudo-clases en CSS son palabras clave que se agregan a los selectores para especificar un estado especial del elemento seleccionado. Permiten aplicar estilos a elementos en función de su estado o posición en el documento.

  • :hover: se aplica cuando el usuario pasa el cursor sobre un elemento.
  • :focus: se aplica cuando un elemento recibe el foco, como al hacer clic en un campo de formulario.
  • :active: se aplica al elemento activo. (por ejemplo, un botón presionado)
