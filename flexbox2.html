Alinear y centrar en Flexbox

En Flexbox, puedes alinear y centrar elementos de manera fácil utilizando las propiedades de alineación. Aquí te mostramos cómo hacerlo:

  • Justify-content: esta propiedad se utiliza para alinear los elementos a lo largo del eje principal (horizontal). Puedes usar valores como center, flex-start, flex-end, space-between, space-around, entre otros.
  • Align-items: esta propiedad se utiliza para alinear los elementos a lo largo del eje transversal (vertical). Puedes usar valores como center, flex-start, flex-end, stretch, entre otros.
  • Align-self: esta propiedad se utiliza para alinear un elemento específico a lo largo del eje transversal (vertical). Puedes usar los mismos valores que align-items.
